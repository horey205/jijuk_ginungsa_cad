# 📝 지적기능사 실기 지원 프로그램 개발 기획서

## 1. 개요
본 프로그램은 지적기능사 작업형 시험의 핵심인 **도곽선/경계점 계산**과 **면적 측정부 작성** 과정을 자동화하고 보조하여, 수험생의 계산 실수를 줄이고 AutoCAD 작업 효율을 극대화하는 것을 목적으로 함.

---

## 2. 도곽선 사이즈 및 실세계 면적 (축척별)

시험 문제에 따라 아래 표의 실세계 면적(m)을 기준으로 도곽선을 결정함.

## 3. 주요 기능 및 프로세스

### 🟢 [Step 1] 도면 생성 및 준비 (AutoCAD 연동)
사용자가 입력한 데이터를 기반으로 AutoCAD에서 즉시 작업 가능한 DXF 파일을 생성함.

*   **도곽선 자동 계산 (①②③④ 법칙 - 우측 상단 기준):**
    *   **원점 설정:** 
        *   지역좌표: X=500,000 / Y=200,000
        *   세계좌표: X=600,000 / Y=200,000
    *   **사분면 대응:** 모든 사분면(1~4사분면)에서 정수 단위 도곽이 형성되도록 내림(`floor`) 로직 적용.
*   **DXF 레이어 표준화:**
    *   `01_도곽선`: 하늘색(Cyan), 실선
    *   `02_경계선`: 적색(Red), 실선
    *   `03_텍스트`: 지번, 좌표값 등 레이어 분리
*   **좌표점 자동 플롯:**
    *   방위각과 거리를 이용해 계산된 경계점들을 도면에 미리 배치하여 수작업 입력 생략.

### 🔵 [Step 2] 면적 측정 및 보정 (지적측량규칙 반영)
AutoCAD에서 측정한 면적값을 입력받아 법적 기준에 맞는 최종 결정 면적을 산출함.

*   **면적 오차 배분 로직:**
    *   `원면적` vs `측정면적 합계` 비교를 통한 오차($e$) 산출.
    *   오차 배분 방식: 각 필지의 면적 크기에 비례하여 배분 ($e \times \frac{a_n}{\sum a}$).
*   **단수 처리 (5사5입):**
    *   지적측량시행규칙에 따른 처리 로직 자동화.
    *   0.5 미만 버림, 0.5 초과 올림, 정확히 0.5일 때 구하려는 끝자리가 짝수면 버리고 홀수면 올림.
*   **결과 테이블 시각화:**
    *   측정면적, 보정계수, 보정면적, 결정면적을 표 형태로 출력.

## 4. 핵심 계산 수식 (Logic)

### A. 도곽선 결정 (이미지 필기법 기준)
이미지 예시 (4118점: X 453478.70, Y 193664.65 / 1/1200 축척) 기반 로직:

1.  **원점 차이 계산:**
    *   $dx = x_{input} - 500,000$
    *   $dy = y_{input} - 200,000$
2.  **단위 도곽 배수 산출 (내림):**
    *   $n_x = \lfloor dx / \text{종선지상길이} \rfloor$
    *   $n_y = \lfloor dy / \text{횡선지상길이} \rfloor$
3.  **우측 상단(P2) 좌표 결정:**
    *   $x_2 = (n_x + 1) \times \text{종선지상길이} + 500,000$
    *   $y_2 = (n_y + 1) \times \text{횡선지상길이} + 200,000$
4.  **좌측 하단(P1) 좌표 결정:**
    *   $x_1 = x_2 - \text{종선지상길이}$
    *   $y_1 = y_2 - \text{횡선지상길이}$

> **[참고]** 4사분면 등 음수 영역에서도 `floor` 함수를 사용하면 정확하게 현재 위치를 포함하는 하단 배수가 계산되므로, 거기에 1단위를 더해 우측 상단을 먼저 잡는 방식이 안전함.

### B. 좌표 산출 (방위각/거리)
*   $X_{new} = X_{known} + (\text{거리} \times \cos(\text{방위각}))$
*   $Y_{new} = Y_{known} + (\text{거리} \times \sin(\text{방위각}))$

## 5. UI/UX 구성안 (6단계 프로세스)

프로그램은 지적기능사 시험 순서에 맞춘 6개의 탭으로 구성함.

1.  **[Tab 1] 도곽선 결정**
    *   입력: 축척(Combo), 기준점 좌표(X, Y)
    *   출력: 좌하단(x1, y1) 및 우상단(x2, y2) 좌표
    *   기능: DXF 도곽선 및 십자선 생성 내보내기

2.  **[Tab 2] 필지 경계선 산출**
    *   입력: 기지점 좌표, 방위각(도/분/초), 거리
    *   출력: 신규 경계점 좌표 (P1, P2...)
    *   기능: 방위각 삼각함수 계산 및 DXF 선분 추가

3.  **[Tab 3] 분할 준비 좌표**
    *   입력: 분할 조건 (거리, 비율 등)
    *   출력: 분할선상의 보조 좌표점

4.  **[Tab 4] 교차점(최종분할점) 산출**
    *   입력: 두 직선의 네 점 좌표 (직선 A: P1-P2, 직선 B: P3-P4)
    *   출력: 두 직선의 교점(Intersection) 좌표
    *   기능: 직선 방정식을 이용한 교점 자동 계산

5.  **[Tab 5] 분할 면적 계산**
    *   입력: 각 필지를 구성하는 점들의 좌표 리스트
    *   출력: 좌표면적계산법에 의한 필지별 측정면적
    *   기능: 신발끈 공식(Shoelace formula)을 이용한 정밀 면적 산출

6.  **[Tab 6] 면적 측정부 작성**
    *   입력: 원면적, 필지별 측정면적(Tab 5 연동)
    *   출력: 보정계수, 산후면적, 결정면적
    *   기능: 5사5입 적용 및 오차 배분 로직
