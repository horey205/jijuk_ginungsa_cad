제공해주신 2017년 지적기능사 시험 문제와 파이썬 코드를 결합하여, 지적기능사 실기 시험의 핵심 계산 과정을 자동화하고 DXF로 내보내는 프로그램 구조를 제안해 드립니다.지적기능사 작업형 시험은 크게 도곽선 자격 계산, 경계점(방위각/거리) 계산, 분할점 계산, 면적 측정으로 나뉩니다. 이 로직을 프로그램에 녹여내면 실무 및 학습용 툴로 매우 유용할 것입니다.  1. 지적기능사 맞춤형 GUI 프로그램 설계시험 문제의 데이터를 기반으로, 다음과 같은 계산 로직을 포함하는 것이 좋습니다.핵심 계산 로직 (수학적 원리)방위각과 거리를 이용한 좌표 계산 ($dx, dy$):$$X_{new} = X_{known} + (Distance \times \cos(Azimuth))$$$$Y_{new} = Y_{known} + (Distance \times \sin(Azimuth))$$시험 문제에서는 소수점 둘째 자리까지 5사5입(반올림의 일종)을 적용해야 합니다.  도곽선 좌표 구획:
주어진 기준점 좌표를 400m(축척 1/1200 기준 단종선/단횡선 간격) 단위로 정수화하여 상하좌우 도곽 수치를 결정합니다.  2. 확장된 프로토타입 코드 (Python + ezdxf)이 코드는 사용자가 기준점(4119) 좌표와 방위각, 거리를 입력하면 필지의 경계점을 계산하고 이를 DXF 파일로 도면화하는 예시입니다.Pythonimport tkinter as tk
from tkinter import messagebox, filedialog
import ezdxf
import math

def calculate_and_export():
    try:
        # 1. 기본 데이터 입력 (기준점 4119 좌표) [cite: 34]
        base_x = 453405.55
        base_y = 193726.90
        
        # 2. 사용자로부터 방위각(도)과 거리 입력받기
        angle_deg = float(entry_angle.get())
        distance = float(entry_dist.get())
        
        # 3. 좌표 계산 로직 (라디안 변환 및 삼각함수 적용) [cite: 51, 52]
        angle_rad = math.radians(angle_deg)
        new_x = base_x + (distance * math.cos(angle_rad))
        new_y = base_y + (distance * math.sin(angle_rad))
        
        # 4. DXF 생성
        doc = ezdxf.new('R2010')
        msp = doc.modelspace()
        
        # 기준점과 계산된 경계점 연결 (선 그리기)
        msp.add_line((base_y, base_x), (new_y, new_x)) # CAD 좌표계 반영 (Y, X 순서 주의)
        
        # 점 번호 텍스트 추가 [cite: 101]
        msp.add_text("P1", dxfattribs={'height': 2.0}).set_placement((new_y, new_x))
        
        # 5. 파일 저장
        save_path = filedialog.asksaveasfilename(defaultextension=".dxf")
        if save_path:
            doc.saveas(save_path)
            messagebox.showinfo("완료", f"경계점 ({new_x:.2f}, {new_y:.2f}) 계산 및 DXF 저장 완료")
            
    except ValueError:
        messagebox.showerror("오류", "숫자 형식을 확인해주세요.")

# GUI 구성
root = tk.Tk()
root.title("지적기능사 좌표 계산기")

tk.Label(root, text="방위각 (Degree):").grid(row=0, column=0)
entry_angle = tk.Entry(root)
entry_angle.insert(0, "88.787") # 문제의 88°47'13" 예시 [cite: 33]
entry_angle.grid(row=0, column=1)

tk.Label(root, text="거리 (m):").grid(row=1, column=0)
entry_dist = tk.Entry(root)
entry_dist.insert(0, "119.99") # 문제 예시 거리 [cite: 33]
entry_dist.grid(row=1, column=1)

tk.Button(root, text="DXF 내보내기", command=calculate_and_export).grid(row=2, columnspan=2)

root.mainloop()
